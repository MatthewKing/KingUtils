# KingUtils

A collection of miscellaneous extension methods and utility classes.

## Installation

Just grab it from [NuGet](https://www.nuget.org/packages/KingUtils/)

```
PM> Install-Package KingUtils
```

```
$ dotnet add package KingUtils
```
